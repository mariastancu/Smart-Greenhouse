Welcome to the Smart Greenhouse Application!

This web application is designed to monitor and optimize the conditions in your greenhouse using modern technology. With the help of this platform, you can ensure that your plants grow in the best possible conditions.

How it works:

Data Collection

Sensors installed in the greenhouse collect the necessary data and send it to the Arduino Uno R4.

Data Transmission

The data is then transmitted in real-time to the Firebase Realtime Database.

Visualization

Users can view this data on the web application, either in real-time or historically, to make informed decisions about managing the greenhouse.
